# 整体介绍

可以看到Autoware项目目录如下，src文件夹为主要的项目代码

* core为项目的基础通用模块
  * msgs 文件为消息传递文件
* launcher启动文件
* universe 七大组件、系统组件以及一些其他文件

本文档将详细解释autoware\_msgs和launcher
